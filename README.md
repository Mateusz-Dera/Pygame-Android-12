# Pygame-Android-12
## Info:
- Pygame project template
- Android 12 __API 31__
- Instalation script
- Build & run script
- Easy to modify __AndroidManifest.tmpl.xml__ file
- Disabled app split screen 
``` 
android:resizeableActivity="false"
```
- Android adaptive __Material You__  icons

## Installation:
#### Ubuntu 21.10
```bash
install.sh
```

## Build & run:
#### Ubuntu 21.10
```bash
build.sh
```
