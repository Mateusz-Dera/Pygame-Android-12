# Pygame-Android-12
## Info:
Pygame buildozer project template
Tested on Ubuntu 21.10
Default Android 12 (API 31)

## Installation
```bash
install.sh
```

## Build & run project
```bash
build.sh
```

## Features
- Instalation script
- Build & run script
- Separated AndroidManifest.tmpl.xml
- android:exported="true"
- Disabled split screen (android:resizeableActivity="false")
